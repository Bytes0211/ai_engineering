# Example Content for Prompt Templates

This file contains example content for each prompt template defined in `prompt_templates.json`. These examples demonstrate the type of input each template is designed to handle.

## Business Communication

```
Hey team, can we hurry up on the report? Its taking too long.
```

## Customer Support & Service

```
My package was supposed to arrive Tuesday, but tracking hasn't updated in three days.
```

## Data Analysis & Insights

```
Summarize the key insights from this dataset and explain what actions the business should consider.
```

## Strategy, Planning & Decision Support

```
We are deciding whether to expand into a new region. Provide strategic options and risks.
```

## HR, People, and Leadership

```
You need to stop missing deadlines. It's becoming a problem.
```

## Marketing, Sales & Customer Messaging

```
Our software helps teams work together.
```

## Documentation, SOPs & Knowledge Base

```
Employees submit a request, managers approve it, finance processes payment.
```

## Product, Engineering & Technical Work

```
Users should be able to export their data.
```

## Compliance, Risk & Governance

```
Employees must adhere to data retention requirements outlined in section 4.2.
```

## Executive-Level Communication

```
We are behind schedule due to vendor delays, but mitigation steps are underway.
```
